# BubbleBreak  
Ice-breaking Web App for Finding Common Interests

A lightweight web application designed to help people break the ice by discovering shared interests through an interactive bubble-popping interface.

---

# 🇺🇸 English

## Overview

**BubbleBreak** is a simple web-based icebreaking tool that helps users discover common interests in an engaging way.

Instead of awkward introductions, users interact with bubbles representing different topics.  
By popping bubbles, participants can reveal shared interests and start conversations naturally.

This project focuses on building a **simple interactive user experience** for social situations such as:

- team building sessions
- first meetings
- networking events
- classroom icebreakers

---

## Key Features

- Interactive bubble-popping UI
- Topic-based conversation starters
- Lightweight front-end architecture
- Simple and responsive user interface

# 🇰🇷 한국어

## 프로젝트 개요

**BubbleBreak**는 사람들이 서로 어색한 상황에서  
**공통 관심사를 발견하며 자연스럽게 대화를 시작할 수 있도록 돕는 웹 서비스**입니다.

사용자는 다양한 주제를 담고 있는 **버블을 터뜨리는 인터랙션**을 통해  
공통 관심사를 발견하고 대화를 시작할 수 있습니다.

이 프로젝트는 다음과 같은 상황에서 활용될 수 있습니다.

- 팀 빌딩 활동
- 첫 만남 및 네트워킹
- 학교 오리엔테이션
- 워크숍 및 세미나
